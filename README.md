## 🛠️ Tech Stack

This portfolio is built with modern web technologies:

- **Frontend Framework**: Vanilla JavaScript with Three.js for 3D graphics
- **3D Graphics**: Three.js for creating immersive 3D experiences
- **Styling**: Pure CSS with modern features like CSS Grid and Flexbox
- **Responsive Design**: Mobile-first approach with media queries
- **Performance**: Optimized assets and lazy loading techniques

## 🧩 Project Structure

```
cyberfolio/
├── assets/
│   └── images/          # All image assets
├── .vscode/            # VS Code configuration
├── index.html          # Main HTML file
├── style.css          # Main stylesheet
├── script.js          # Main JavaScript file with 3D animations
└── README.md          # Project documentation
```

The project follows a clean and organized structure:
- `assets/` contains all static resources like images
- `index.html` is the main entry point with semantic HTML5 structure
- `style.css` handles all styling with modern CSS features
- `script.js` contains the core functionality including:
  - 3D grid generation and animations
  - Interactive navigation
  - Responsive sidebar functionality
  - Smooth scrolling behavior

## 🎨 Customization Guide

Make this portfolio truly yours with these customization options:

### 🎯 Basic Customizations
- **Personal Information**: Edit the content in `index.html` to add your name, bio, and social links
- **Color Scheme**: Modify the color variables in `style.css` to match your brand
- **Images**: Replace the images in `assets/images/` with your own photos and graphics
- **Projects**: Update the projects section with your own work and achievements

### 🛠️ Advanced Customizations
- **3D Grid**: Adjust the grid parameters in `script.js`:
  - Change `cellSize` and `size` in the `HexGrid` configuration
  - Modify animation timings and effects
  - Customize the grid colors and materials
- **Layout**: Modify the CSS in `style.css` to:
  - Change the layout structure
  - Adjust spacing and typography
  - Add new sections or components
- **Animations**: Fine-tune the animations by:
  - Adjusting TWEEN.js parameters
  - Modifying Three.js camera movements
  - Customizing hover effects

### 💡 Tips for Customization
1. **Start Small**: Begin with basic content changes before diving into code
2. **Test Responsively**: Always check how changes look on different devices
3. **Keep Performance in Mind**: Optimize images and animations for smooth performance
4. **Backup First**: Make a copy of the original files before making major changes
